darwin_webkit exposes bindings to some of the WebKit's API on MacOS for Rust.

Modules follow the cocoa crate convention for naming bindings.

It uses the objc and cocoa crates to bind with Objective-C.

The darwin_webkit::foundation module exposes some dependencies to using the WKWebView APIs, like NSURLRequest.

The darwin_webkit::webkit module exposes bindings to the WKWebView API.

In darwin_webkit::helpers there's a very small higher level wrapper that may turn into a higher level API.
